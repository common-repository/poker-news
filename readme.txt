A simple widget which displays a configurable list of the latest Poker News.

== Description ==

This plugin fetches all relevant publicly available news feeds of poker news. The news can be placed on the blog through a highly configurable widget.
The widget allows you to edit the entire formatting of each RSS item, such as description, count, title and even how the list should appear and the link tags themselves.

== Installation ==

1. Upload the folder `poker-news` to the `/wp-content/plugins/` directory or search for `Poker News` in `Plugins` > `Add new` and click `Install` to the right of `Poker News`
2. Activate the plugin through the `Plugins` menu in WordPress
